## Release / post copy

### One-sentence product summary

Obsidian Table Exporter helps you take a table that already looks right in Obsidian and turn it into a clean image, PDF, CSV, or Excel file without rebuilding it elsewhere.

### Key value props to repeat consistently

- long tables do not need manual stitched screenshots
- spreadsheet handoff is built in
- PDF export is designed for row-aware pagination
- the plugin works from rendered tables, not only raw Markdown
