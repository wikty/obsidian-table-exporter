# Code of Conduct

## Our Pledge

We want `Obsidian Table Exporter` to be a friendly project for users, contributors, and maintainers. We expect participation to be respectful, constructive, and focused on making the project better.

## Expected Behavior

- Be respectful in issues, pull requests, discussions, and releases.
- Assume good intent and give actionable feedback.
- Keep technical disagreement focused on the work, not the person.
- Share reproduction steps, screenshots, or sample files when reporting bugs.

## Unacceptable Behavior

- Harassment, insults, or personal attacks
- Discriminatory or exclusionary language
- Repeated bad-faith arguing or derailing discussion
- Publishing someone else's private information without permission

## Scope

This code of conduct applies to project spaces, including GitHub issues, pull requests, release discussions, and other linked community spaces used for the project.

## Enforcement

Project maintainers may edit, remove, or close content that violates these expectations. Repeated or severe violations may lead to a temporary or permanent ban from project spaces.

## Contact

If you need to report a conduct issue privately, open a private security-style report through the repository maintainers if available, or contact the maintainer through the repository profile.
